Class Activity: Dynamic Dropdowns

In this activity, you will use JavaScript to dynamically populate a dropdown using an array and DOM methods.

Expectations

  • All Javascript code should be in a <script> block in the <head> section of the HTML page.
  • Do not use document.write()
  • Use addEventListener() for events
  • Functions are defined using the arrow syntax
  • Use DOMContentLoaded event to make sure content is loaded.

Instructions

  1. Create a JavaScript array containing at least 5 fruit names.
  2. Write a function that:
    • Clears all existing options from the dropdown.
    • Adds a default option labeled "Select One".
    • Loops through the array and creates an <option> for each item.
  3. Use createElement() and appendChild() to build the dropdown.
  4. Add an event listener to the button so the dropdown loads when clicked.
