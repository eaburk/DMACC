Functions Demonstration

Instructions:

  • Create two global variables - petName and animalType. Set their values to whatever you'd like.
  • Create a function called displayPet() that uses alert() to display the name and animal type. Call it during runtime execution.
  • Create a function called getPetDetails().
    • Use the prompt() to set a new value in the global variable petName
    • Use the prompt() to set a new value in the global variable animalType
  • Create a button with a click event to call getPetDetails().
  • Create a button with a click event to call displayPet().
  • Notice when you click the button to display the pet details it will change after entering new pet details.
